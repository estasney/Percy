This page provides insights into the genders present in a spreadsheet of names

Each known name is assigned a minimum and maximum probability of being female, based on the number of male and number of female records

For common names, such as James, the minimum and maximum probabilities are nearly identical

For less common names, such as Anargyros, these are very far apart

Name N_Male N_Female Min Prob Female Max Prob Female
James 5,139,460 23,626 0.45% 0.46%
Anargyros 1 0 1.26% 84.19%

Select a spreadsheet from your computer. .xlsx, .xls, and .csv file types are supported.


Ensure that you have a column where first names are listed. Enter the column name in "Header for First Name"
