This page demonstrates the concept of "word embeddings"


The results returned are words that were found to be used in similar contexts as your lookup word.


Take, for instance, "mongodb". This is a "NoSQL" database. It is one of several fundamental difference that divide database technologies.


If we were to quantify the "relatedness" of say, "mongodb" to "MS SQL Server" and "Apache Cassandra", "Apache Cassandra" should score higher than "MS SQL Server" as these are both "NoSQL".


Some services attempt to do the same quantification through correlation, which can lead your search in the wrong direction. See what HiringSolved returns for this same search.
