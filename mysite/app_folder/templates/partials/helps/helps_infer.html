This page provides an insight into the mechanics of Diversity Score


After entering a first name of your choosing, click "Guess Gender" to see the results


Now try some variations of the same name. Have you found one not present in our name database?


This is where the machine learning model takes over. It looks for patterns, not exact matches and so can provide reasonably accurate guesses.


The model is optimized for Anglo-Saxon names, i.e. John, Jane, etc. This is a result of the lack of available name and gender data available globally.
